# 原创美术替代实施单

> 这份实施单的目标不是“搬资源”，而是把无授权参考包拆成可执行的原创生产任务。
> 参考的是气质、层次、功能组织，不是具体图片本身。

## 1. 风格锚点

- 目标气质：`Q版水墨国风 + 修仙器物感 + 轻灵气流光`
- 关键词：
  - 山门
  - 云海
  - 墨色
  - 铜纹
  - 卷轴
  - 符箓
  - 灵火
- 视觉边界：
  - 不做手游大红大金堆满屏
  - 不做写实仙侠立绘
  - 不做西式奇幻皮革 UI
  - 保持留白、层次、装饰克制

## 2. 从参考包里“学什么”

- 学整体结构：
  - 面板分层
  - 标题牌匾
  - 按钮层级
  - 稀有度表达
  - 飘字节奏
- 学氛围处理：
  - 水墨底纹
  - 暗底金边
  - 青玉绿高亮
  - 危险状态暗红处理
- 学功能覆盖面：
  - 通用按钮
  - 页签
  - 提示框
  - 锁定图标
  - 资源条

## 3. 明确不能照搬的内容

- 具体按钮底图
- 具体边框形状
- 具体角色立绘
- 具体敌人图
- 具体 icon
- 具体特效序列
- 具体字体文件

## 7. 资源生成提示词方向

- UI：
  - `Q版水墨国风修仙游戏UI，深色漆木与旧铜边框，青玉绿高亮，卷轴式信息面板，干净留白，非现代科幻`
- 敌人：
  - `Q版修仙敌人立绘，半身像，水墨边缘，低饱和布衣，带邪气符纹，适合2D RPG 战斗界面`
- 背景：
  - `东方修仙山门场景，云海、石阶、牌楼、淡墨远山，适合游戏主菜单背景，留出中间UI区域`
- 特效：
  - `水墨斩击、灵火爆点、金色灵纹闪光，透明背景，适合2D游戏特效贴图`

## 8. 下一步落地建议

1. 先做 UI 基础件，不然每个界面都会重复返工。
2. 再做历练敌人占位立绘和命中特效，收益最高。
3. 然后回头统一主菜单和世界地图的视觉皮肤。
