# UI 美术资源生成说明

> 约束补充：无授权第三方资源只能放在项目根目录 `Reference/` 作为风格参考，不允许直接进入 `Assets/` 作为正式游戏资源。
> 正式生产目录见 `GAME_ASSET_DIRECTORY_GUIDE.md`，原创替代计划见 `ORIGINAL_ART_REPLACEMENT_PLAN.md`。

> 这份文档用于给 AI 生成整套游戏界面资源。
> 目标不是“做几张好看的图”，而是给整个项目建立一套统一、可量产、可扩展的 UI 资源风格和分界面需求。

## 1. 项目总说明

### 游戏类型

- 修仙题材单人 RPG
- 结构上是 `主菜单 -> 世界地图 -> 历练远征 -> 事件/战斗 -> 结算 -> 回到大地图`
- 玩法融合：
- 修仙养成
- 地图推进
- 轻 Rogue-like 房间远征
- 资源收集与装备成长
- 门派驻地经营/宗门行动

### 目标平台与显示基线

- 平台：PC
- 主设计分辨率：`1920x1080`
- 兼容比例：`16:9`、`16:10`、`21:9`
- UI 原则：全屏只能有一个主 UI，其他只能是弹窗

### 统一美术风格描述

- 题材关键词：`东方修仙`、`山海志怪`、`宗门洞府`、`灵气`、`符箓`、`古卷`、`青铜器纹样`
- 整体气质：`沉稳、神秘、克制、雅致、带灵性流光`
- 视觉方向：不是二次元糖水 UI，也不是纯写实历史 UI，而是“古风器物感 + 灵气幻想感”的中间路线
- 材质感：
- 主体面板像 `漆木、旧铜、玄石、卷轴、丝绢`
- 高亮和稀有状态用 `金纹、玉光、灵火、雾气`
- 装饰应以 `云纹、剑纹、八卦、阵纹、符纸、古篆印章` 为主
- 色彩体系：
- 主底色：墨黑、青灰、乌木褐、深藏蓝
- 主强调色：鎏金、旧金、青玉绿、丹砂红
- 功能色：
- 正向/可用：青玉绿
- 危险/敌对：暗赤、血棕、紫黑
- 中立/说明：灰金、暖白
- 稀有度建议：
- 凡阶：灰褐
- 灵阶：青绿
- 玄阶：蓝紫
- 地阶/更高阶预留：赤金、月白金
- 光效要求：
- 少量灵气描边
- 边角淡发光
- 按钮高亮有“灵纹点亮”感
- 不要赛博霓虹，不要西式科幻 HUD
- 字体气质：
- 标题适合古风书卷感字体
- 正文适合清晰的宋体/黑体风格
- 数值和按钮文案要清楚、可读、利落

### 统一 UI 语言

- 形状：以 `横向长牌匾`、`卷轴窗`、`古匣卡片`、`印章标签` 为核心造型
- 面板边框：厚薄分级明确，主界面边框厚，次级信息框薄
- 图标风格：统一为“半写实图标 + 东方器物造型 + 控制细节数量”
- 按钮风格：
- 普通按钮：木牌/铜牌感
- 重要确认按钮：金纹强化
- 危险操作按钮：暗赤边框
- 禁用按钮：褪色灰化
- 弹窗：必须有遮罩层，遮罩偏暗蓝黑，不要纯黑

## 1.5 资源尺寸与导出硬规范

> 后续所有 AI 生成都先按这里的尺寸出图，再做裁切或缩放，不再临时猜尺寸。
> 当前项目 UI 设计基线来自现有 prefab 与 runtime builder，统一以 `1920x1080` 为主画布。

### 全局导出规则

- 全屏背景最终交付尺寸：`1920x1080`
- 半屏或大面板插图最终交付尺寸：长边不超过 `1536`
- UI 按钮、边框、面板底图默认输出 `PNG`
- 全屏背景默认输出 `PNG` 或 `JPG`
- 需要透明背景的 UI 资源必须输出 `PNG`
- 需要九宫格的资源，四边必须预留干净边距，避免角花被拉伸
- 默认像素密度按 `1x` 交付，不额外生成 2x，避免当前项目资源体积膨胀
- 图片命名统一小写下划线，不带空格，不带中文

### 九宫格与留边规则

- 大面板底图：
  - 角区安全边距：`48px`
  - 上下边安全边距：`32px`
  - 左右边安全边距：`32px`
- 中型弹窗底图：
  - 角区安全边距：`36px`
  - 上下边安全边距：`24px`
  - 左右边安全边距：`24px`
- 按钮底图：
  - 角区安全边距：`18px`
  - 水平安全边距：`24px`
  - 垂直安全边距：`14px`
- 小型 icon 底板：
  - 四边安全边距：`8px`

## 4. 推荐给 AI 的总提示词

> 下面这一段可以作为总风格前置提示，每次生成单页资源时都带上。

### 中文版

为一款东方修仙题材单人 RPG 设计 UI 资源。游戏包含主菜单、世界地图、历练远征、事件卡、战斗 HUD、背包、工坊、门派驻地等界面。整体风格要求古风仙侠、山海志怪、沉稳神秘、雅致克制，不要现代科幻 HUD，不要甜腻二次元，也不要纯历史写实。UI 材质以漆木、旧铜、玄石、卷轴、丝绢为主，配合少量灵气流光、阵纹、符箓、云纹、玉石高光。主色调为墨黑、青灰、乌木褐、藏蓝，强调色为旧金、青玉绿、丹砂红。所有按钮和卡片都需要明确的默认态、悬停态、按下态、禁用态和选中态。所有界面必须保证层级清晰、信息可读、功能控件明确。

### 简版关键词

- 东方修仙
- 山海志怪
- 古风 UI
- 漆木旧铜
- 玉石灵气
- 卷轴面板
- 阵纹符箓
- 沉稳雅致
- 神秘克制
- PC RPG HUD

## 5. 推荐生成顺序

1. 先生成通用控件库
2. 再生成主菜单全套
3. 再生成世界地图和背包/工坊
4. 再生成历练 HUD、事件弹窗、暂停弹窗
5. 最后补通用 Tooltip、Toast、确认窗、结算页
